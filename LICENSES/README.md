# License references

- Linux kernel licensing and the GPL-2.0-only text are distributed in a Linux
  source tree under `COPYING` and `LICENSES/preferred/GPL-2.0`.
- Mesa's license information is distributed in a Mesa source tree under
  `docs/license.rst`.

Those upstream texts govern the preserved source-derived patches; the
top-level MIT license does not replace them.  Exact patch bytes are retained
for test provenance, so license identifiers were not inserted into their mail
payloads.
